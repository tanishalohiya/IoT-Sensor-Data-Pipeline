# IoT Sensor Data Pipeline on Cloud

## Overview
This project simulates IoT sensor data (temperature & humidity) and sends it to the cloud using MQTT. Data is processed using Node-RED and visualized on a live dashboard.

## Tech Stack
- Python (Sensor Simulation)
- HiveMQ (MQTT Broker)
- Node-RED (Processing)
- Dashboard UI

## Features
- Real-time data streaming
- Live graphs and gauges
- Alert system
- Data storage in file

## How to Run
1. Run Node-RED
2. Run sensor.py


## Output
- Live dashboard with graphs and alerts
